# 一、产品概述

数字医生平台是一个支持 **医疗知识管理 + AI数字医生问诊 + 机构协作** 的系统。

核心目标：

1. 医疗知识管理（文件 + 知识卡）
2. AI数字医生问诊体验
3. 医生数字分身管理
4. 机构 / 科室协同知识库
5. API能力开放

---

# 二、角色与权限

| 角色 | 权限 |
|---|---|
| 系统管理员 | 全局管理 |
| 机构管理员 | 管理机构、科室、成员 |
| 科室管理员 | 管理科室成员与资源 |
| 普通用户 | 使用系统功能 |

权限层级：

系统管理员 > 机构管理员 > 科室管理员 > 普通用户

# 五、边界条件

系统不负责：

- 医疗诊断责任
- 医疗合规审核

限制：

1. 单文件最大 100MB
2. 批量上传最大 100个
3. Token最大数量 50/用户
