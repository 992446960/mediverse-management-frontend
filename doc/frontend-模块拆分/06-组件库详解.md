# 组件库详解

## 1. UI 基础组件 (shadcn/ui)

位置: `frontend/components/ui/`

### 表单组件

#### Input

**文件**: `input.tsx`  
**用途**: 文本输入框  
**变体**: 默认样式  
**示例**:

```tsx
<Input placeholder="请输入..." />
<Input type="password" />
<Input disabled />
```

#### Textarea

**文件**: `textarea.tsx`  
**用途**: 多行文本输入  
**示例**:

```tsx
<Textarea placeholder='请输入描述...' className='min-h-[100px]' />
```

#### Select

**文件**: `select.tsx`  
**用途**: 下拉选择框  
**组件**: SelectTrigger, SelectContent, SelectItem, SelectValue  
**示例**:

```tsx
<Select value={value} onValueChange={setValue}>
  <SelectTrigger>
    <SelectValue placeholder='请选择' />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value='option1'>选项1</SelectItem>
    <SelectItem value='option2'>选项2</SelectItem>
  </SelectContent>
</Select>
```

#### RadioGroup

**文件**: `radio-group.tsx`  
**用途**: 单选按钮组  
**组件**: RadioGroup, RadioGroupItem  
**示例**:

```tsx
<RadioGroup value={value} onValueChange={setValue}>
  <div className='flex items-center space-x-2'>
    <RadioGroupItem value='option1' id='opt1' />
    <Label htmlFor='opt1'>选项1</Label>
  </div>
</RadioGroup>
```

#### Label

**文件**: `label.tsx`  
**用途**: 表单标签  
**示例**:

```tsx
<Label htmlFor="name">名称</Label>
<Input id="name" />
```

#### Form

**文件**: `form.tsx`  
**用途**: 表单容器（集成 React Hook Form）  
**组件**: Form, FormField, FormItem, FormLabel, FormControl, FormMessage

### 数据展示组件

#### Table

**文件**: `table.tsx`  
**用途**: 数据表格  
**组件**: Table, TableHeader, TableBody, TableRow, TableHead, TableCell  
**示例**:

```tsx
<Table>
  <TableHeader>
    <TableRow>
      <TableHead>列1</TableHead>
      <TableHead>列2</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>数据1</TableCell>
      <TableCell>数据2</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

#### Card

**文件**: `card.tsx`  
**用途**: 卡片容器  
**组件**: Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter  
**示例**:

```tsx
<Card>
  <CardHeader>
    <CardTitle>标题</CardTitle>
    <CardDescription>描述</CardDescription>
  </CardHeader>
  <CardContent>内容</CardContent>
  <CardFooter>底部</CardFooter>
</Card>
```

#### Badge

**文件**: `badge.tsx`  
**用途**: 标签徽章  
**变体**: default, secondary, destructive, outline  
**示例**:

```tsx
<Badge>默认</Badge>
<Badge variant="secondary">次要</Badge>
<Badge variant="destructive">危险</Badge>
<Badge variant="outline">轮廓</Badge>
```

#### Avatar

**文件**: `avatar.tsx`  
**用途**: 用户头像  
**组件**: Avatar, AvatarImage, AvatarFallback  
**示例**:

```tsx
<Avatar>
  <AvatarImage src='/avatar.jpg' alt='用户' />
  <AvatarFallback>张三</AvatarFallback>
</Avatar>
```

### 交互组件

#### Button

**文件**: `button.tsx`  
**用途**: 按钮  
**变体**: default, destructive, outline, secondary, ghost, link  
**尺寸**: default, sm, lg, icon  
**示例**:

```tsx
<Button>默认按钮</Button>
<Button variant="outline">轮廓按钮</Button>
<Button size="sm">小按钮</Button>
<Button variant="ghost" size="icon">
  <Icon />
</Button>
```

#### Dialog

**文件**: `dialog.tsx`  
**用途**: 对话框/模态框  
**组件**: Dialog, DialogTrigger, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogFooter  
**示例**:

```tsx
<Dialog open={isOpen} onOpenChange={setIsOpen}>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>标题</DialogTitle>
      <DialogDescription>描述</DialogDescription>
    </DialogHeader>
    <div>内容</div>
    <DialogFooter>
      <Button>确定</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

#### AlertDialog

**文件**: `alert-dialog.tsx`  
**用途**: 警告确认对话框  
**组件**: AlertDialog, AlertDialogTrigger, AlertDialogContent, AlertDialogHeader, AlertDialogTitle, AlertDialogDescription, AlertDialogFooter, AlertDialogAction, AlertDialogCancel  
**示例**:

```tsx
<AlertDialog open={isOpen} onOpenChange={setIsOpen}>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>确认删除?</AlertDialogTitle>
      <AlertDialogDescription>此操作无法撤销。</AlertDialogDescription>
    </AlertDialogHeader>
    <AlertDialogFooter>
      <AlertDialogCancel>取消</AlertDialogCancel>
      <AlertDialogAction onClick={handleDelete}>确认删除</AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```

#### DropdownMenu

**文件**: `dropdown-menu.tsx`  
**用途**: 下拉菜单  
**组件**: DropdownMenu, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuItem  
**示例**:

```tsx
<DropdownMenu>
  <DropdownMenuTrigger asChild>
    <Button variant='ghost' size='icon'>
      <MoreHorizontal />
    </Button>
  </DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuItem>编辑</DropdownMenuItem>
    <DropdownMenuItem>删除</DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>
```

#### Collapsible

**文件**: `collapsible.tsx`  
**用途**: 可折叠容器  
**组件**: Collapsible, CollapsibleTrigger, CollapsibleContent  
**示例**:

```tsx
<Collapsible open={isOpen} onOpenChange={setIsOpen}>
  <CollapsibleTrigger>
    <Button variant='ghost'>
      {isOpen ? <ChevronDown /> : <ChevronRight />}
      展开/折叠
    </Button>
  </CollapsibleTrigger>
  <CollapsibleContent>内容</CollapsibleContent>
</Collapsible>
```

### 容器组件

#### Sidebar

**文件**: `sidebar.tsx`  
**用途**: 侧边栏容器  
**组件**: Sidebar, SidebarProvider, SidebarTrigger, SidebarHeader, SidebarContent, SidebarFooter, SidebarGroup, SidebarGroupLabel, SidebarMenu, SidebarMenuItem, SidebarMenuButton, SidebarMenuSub, SidebarMenuSubItem, SidebarMenuSubButton, SidebarRail

#### ScrollArea

**文件**: `scroll-area.tsx`  
**用途**: 滚动区域  
**示例**:

```tsx
<ScrollArea className='h-[400px]'>
  <div>长内容...</div>
</ScrollArea>
```

#### Tabs

**文件**: `tabs.tsx`  
**用途**: 标签页  
**组件**: Tabs, TabsList, TabsTrigger, TabsContent  
**示例**:

```tsx
<Tabs value={activeTab} onValueChange={setActiveTab}>
  <TabsList>
    <TabsTrigger value='tab1'>标签1</TabsTrigger>
    <TabsTrigger value='tab2'>标签2</TabsTrigger>
  </TabsList>
  <TabsContent value='tab1'>内容1</TabsContent>
  <TabsContent value='tab2'>内容2</TabsContent>
</Tabs>
```

#### Sheet

**文件**: `sheet.tsx`  
**用途**: 侧滑面板  
**组件**: Sheet, SheetTrigger, SheetContent, SheetHeader, SheetTitle, SheetDescription

#### Separator

**文件**: `separator.tsx`  
**用途**: 分隔线  
**示例**:

```tsx
<Separator />
<Separator orientation="vertical" />
```

### 反馈组件

#### Sonner (Toast)

**文件**: `sonner.tsx`  
**用途**: 消息通知  
**使用**:

```tsx
import { toast } from 'sonner'

toast.success('操作成功')
toast.error('操作失败')
toast.info('提示信息')
```

#### Tooltip

**文件**: `tooltip.tsx`  
**用途**: 工具提示  
**组件**: Tooltip, TooltipTrigger, TooltipContent, TooltipProvider  
**示例**:

```tsx
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger>悬停</TooltipTrigger>
    <TooltipContent>提示内容</TooltipContent>
  </Tooltip>
</TooltipProvider>
```

#### Skeleton

**文件**: `skeleton.tsx`  
**用途**: 骨架屏加载占位  
**示例**:

```tsx
<Skeleton className="h-4 w-[250px]" />
<Skeleton className="h-12 w-12 rounded-full" />
```
