# 页面模块详解

## 1. 仪表盘模块 (Dashboard)

**路由**: `/`  
**文件**: `frontend/app/page.tsx`  
**类型**: 服务端组件

### 功能描述

- 显示平台核心统计数据（接诊数、Token 消耗、知识引用等）
- 展示最近活动时间线
- 显示用户配置的数字医生分身列表

### UI 组件使用

- `Card` - 统计卡片容器
- `Badge` - 状态标签
- `Avatar` - 用户头像
- 图标: `Activity`, `Users`, `FileText`, `MessageSquare`, `Lightbulb`, `Zap`

### 数据结构

```typescript
// 统计数据
{
  累计接诊数: number,
  今日接诊数: number,
  累计引用知识: number,
  今日Token消耗: number
}

// 活动记录
{
  type: 'chat' | 'file' | 'knowledge',
  description: string,
  timestamp: string
}
```

### 布局结构

```
┌─────────────────────────────────────────┐
│  统计卡片区域 (4列网格)                  │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐   │
│  │累计  │ │今日  │ │知识  │ │Token │   │
│  │接诊  │ │接诊  │ │引用  │ │消耗  │   │
│  └──────┘ └──────┘ └──────┘ └──────┘   │
├─────────────────────────────────────────┤
│  内容区域 (2列网格)                      │
│  ┌──────────────┐  ┌──────────────┐    │
│  │ 最近活动     │  │ 我的分身     │    │
│  │ (时间线)     │  │ (分身列表)   │    │
│  └──────────────┘  └──────────────┘    │
└─────────────────────────────────────────┘
```

## 2. 聊天模块 (Chat)

**路由**: `/chat`, `/chat/session/[id]`  
**文件**:

- `frontend/app/chat/layout.tsx` (布局)
- `frontend/app/chat/page.tsx` (首页)
- `frontend/app/chat/session/[id]/page.tsx` (对话详情)

### 功能描述

- 选择数字医生分身开始对话
- 查看和管理对话历史
- 进行实时对话交互

### 使用的组件

- `ChatSidebar` - 对话历史侧边栏
- `AvatarSelector` - 分身选择器
- `ThinkingProcess` - 思考过程展示
- `ScrollArea` - 滚动区域

### 状态管理

使用 `useChatStore` (Zustand):

```typescript
{
  sessions: ChatSession[],
  activeSessionId: string | null,
  addSession: (session) => void,
  removeSession: (id) => void,
  renameSession: (id, newTitle) => void,
  setActiveSession: (id) => void
}
```

### 布局结构

```
┌─────────────────────────────────────────┐
│ ChatSidebar │  Main Content             │
│             │                            │
│ [新建对话]  │  ┌──────────────────────┐ │
│             │  │  Avatar Selector     │ │
│ 今天        │  │  (选择数字医生)      │ │
│ - 会话1     │  └──────────────────────┘ │
│ - 会话2     │                            │
│             │  或                        │
│ 昨天        │                            │
│ - 会话3     │  ┌──────────────────────┐ │
│             │  │  Chat Interface      │ │
│ 过去7天     │  │  (对话界面)          │ │
│ - 会话4     │  └──────────────────────┘ │
└─────────────────────────────────────────┘
```

## 3. 知识库模块 (Knowledge Base)

**路由**: `/knowledge-base`, `/knowledge-base/search/[id]`  
**文件**:

- `frontend/app/knowledge-base/layout.tsx` (布局)
- `frontend/app/knowledge-base/page.tsx` (搜索首页)
- `frontend/app/knowledge-base/search/[id]/page.tsx` (搜索结果)

### 功能描述

- 搜索医学知识库（指南、文献、病例、规则）
- 查看搜索历史
- 浏览最新文件和知识卡片

### 使用的组件

- `KnowledgeBaseSidebar` - 搜索历史侧边栏
- `Input` - 搜索输入框
- `Badge` - 标签展示
- `Card` - 推荐内容卡片

### 布局结构

```
┌─────────────────────────────────────────┐
│ KB Sidebar  │  Main Content             │
│             │                            │
│ [新建搜索]  │  ┌──────────────────────┐ │
│             │  │  Search Bar          │ │
│ 今天        │  │  (大搜索框)          │ │
│ - 搜索1     │  └──────────────────────┘ │
│ - 搜索2     │                            │
│             │  最近搜索 (标签云)         │
│ 昨天        │  [标签1] [标签2] [标签3]  │
│ - 搜索3     │                            │
│             │  ┌──────────┐ ┌──────────┐│
│ 过去7天     │  │最新文件  │ │最新知识卡││
│ - 搜索4     │  │(列表)    │ │(列表)    ││
│             │  └──────────┘ └──────────┘│
└─────────────────────────────────────────┘
```
