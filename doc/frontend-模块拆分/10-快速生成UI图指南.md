# 快速生成 UI 图指南

本指南帮助 AI Agent 快速理解项目结构并生成准确的 UI 设计图。

---

## 1. 生成 UI 图的步骤

### 步骤 1: 确定页面类型

根据路由确定页面类型:

| 路由模式            | 页面类型   | 参考文档                             |
| ------------------- | ---------- | ------------------------------------ |
| `/`                 | 仪表盘     | 02-页面模块详解.md                   |
| `/chat/*`           | 聊天模块   | 02-页面模块详解.md                   |
| `/knowledge-base/*` | 知识库     | 02-页面模块详解.md                   |
| `/my/*`             | 个人工作台 | 03-工作台模块.md                     |
| `/dept/*`           | 科室工作台 | 03-工作台模块.md                     |
| `/org/*`            | 机构工作台 | 03-工作台模块.md                     |
| `/admin/*`          | 系统管理   | 04-管理模块.md, 05-API-Token 管理.md |

### 步骤 2: 识别布局结构

**根布局** (所有页面):

```
┌─────────────────────────────────────┐
│ Sidebar │  Header Bar               │
│         │  ┌─────────────────────┐  │
│         │  │                     │  │
│         │  │  Page Content       │  │
│         │  │                     │  │
│         │  └─────────────────────┘  │
└─────────────────────────────────────┘
```

**特殊布局**:

- 聊天/知识库: 左侧历史 + 右侧内容
- 管理页面: 左侧筛选树 + 右侧列表

### 步骤 3: 确定使用的组件

参考 `06-组件库详解.md` 和 `07-业务组件详解.md`

### 步骤 4: 应用设计规范

参考 `09-UI设计规范.md` 中的:

- 颜色系统
- 排版系统
- 间距系统
- 组件样式

## 2. 常见页面模板

### 模板 1: 列表页面 (如机构管理)

**布局**:

```
┌─────────────────────────────────────┐
│  标题                [新增] 按钮    │
├─────────────────────────────────────┤
│  ┌───────────────────────────────┐ │
│  │ 表格                          │ │
│  │ ┌─────┬─────┬─────┬─────────┐│ │
│  │ │列1  │列2  │列3  │ 操作    ││ │
│  │ ├─────┼─────┼─────┼─────────┤│ │
│  │ │数据 │数据 │数据 │ 编辑删除││ │
│  │ └─────┴─────┴─────┴─────────┘│ │
│  └───────────────────────────────┘ │
└─────────────────────────────────────┘
```

**组件清单**:

- `a-table` (表格)
- `a-button` (新增、编辑、删除)
- `a-modal` (新增/编辑对话框)
- `a-popconfirm` (删除确认)
- `a-tag` (状态标签)
- `a-space` (操作按钮组)

**代码示例**:

```vue
<template>
  <div class="page-container">
    <div class="page-header">
      <h2>机构管理</h2>
      <a-button type="primary" @click="handleAdd">
        <template #icon><PlusOutlined /></template>
        新增机构
      </a-button>
    </div>

    <a-card>
      <a-table
        :columns="columns"
        :data-source="dataSource"
        :loading="loading"
        row-key="id"
      >
        <template #bodyCell="{ column, record }">
          <template v-if="column.key === 'status'">
            <a-tag :color="record.status === 'active' ? 'success' : 'default'">
              {{ record.status === 'active' ? '启用' : '禁用' }}
            </a-tag>
          </template>
          <template v-if="column.key === 'action'">
            <a-space>
              <a-button type="link" size="small" @click="handleEdit(record)">
                编辑
              </a-button>
              <a-popconfirm
                title="确定要删除吗？"
                @confirm="handleDelete(record)"
              >
                <a-button type="link" size="small" danger>删除</a-button>
              </a-popconfirm>
            </a-space>
          </template>
        </template>
      </a-table>
    </a-card>
  </div>
</template>
```

### 模板 2: 带侧边栏的列表页面 (如用户管理)

**布局**:

```
┌─────────────────────────────────────┐
│ 侧边栏  │  标题        [新增] 按钮  │
│         ├───────────────────────────┤
│ 筛选树  │  ┌─────────────────────┐ │
│ ▼ 机构1 │  │ 表格                │ │
│   ▶ 科室│  │                     │ │
│ ▼ 机构2 │  └─────────────────────┘ │
└─────────────────────────────────────┘
```

**组件清单**:

- `a-layout`, `a-layout-sider`, `a-layout-content` (布局)
- `a-card` (容器)
- `a-tree` (树形筛选)
- `a-table` (数据列表)
- `a-modal` (对话框)

**代码示例**:

```vue
<template>
  <a-layout style="min-height: calc(100vh - 64px)">
    <!-- 左侧边栏 -->
    <a-layout-sider width="256" theme="light">
      <a-card title="组织架构">
        <a-tree
          :tree-data="treeData"
          :selected-keys="selectedKeys"
          @select="handleTreeSelect"
        />
      </a-card>
    </a-layout-sider>

    <!-- 右侧内容 -->
    <a-layout-content style="padding: 24px">
      <div class="page-header">
        <h2>用户管理</h2>
        <a-button type="primary" @click="handleAdd">
          <template #icon><PlusOutlined /></template>
          新增用户
        </a-button>
      </div>

      <a-card>
        <a-table :columns="columns" :data-source="dataSource" />
      </a-card>
    </a-layout-content>
  </a-layout>
</template>
```

### 模板 3: 配置页面 (如分身配置)

**布局**:

```
┌─────────────────────────────────────┐
│  标题                [测试] 按钮    │
├─────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────┐│
│  │ 配置卡片     │  │ 统计卡片     ││
│  │              │  │              ││
│  │ [头像]       │  │ 累计: 1234   ││
│  │ 名称: [输入] │  │ 今日: 42     ││
│  │ 简介: [文本] │  │ Token: 450K  ││
│  │              │  │              ││
│  │ [保存] 按钮  │  └──────────────┘│
│  └──────────────┘                  │
└─────────────────────────────────────┘
```

**组件清单**:

- `a-card` (卡片容器)
- `a-row`, `a-col` (网格布局)
- `a-avatar` (头像)
- `a-input`, `a-textarea` (表单输入)
- `a-radio-group` (单选)
- `a-tag` (标签)
- `a-button` (操作按钮)
- `a-statistic` (统计数值)

**代码示例**:

```vue
<template>
  <div class="page-container">
    <div class="page-header">
      <h2>我的分身</h2>
      <a-button @click="handleTest">对话测试</a-button>
    </div>

    <a-row :gutter="[16, 16]">

      <a-col :xs="24" :lg="16">
        <a-card title="分身配置">
          <a-form :model="formData" :label-col="{ span: 4 }">

              </a-avatar>
            </a-form-item>
            <a-form-item label="名称">
              <a-input v-model:value="formData.name" />
            </a-form-item>
            <a-form-item label="简介">
              <a-textarea v-model:value="formData.bio" :rows="4" />
            </a-form-item>
            <a-form-item label="沟通风格">
              <a-radio-group v-model:value="formData.style">
                <a-radio value="formal">正式</a-radio>
                <a-radio value="friendly">亲切</a-radio>
                <a-radio value="concise">简洁</a-radio>
              </a-radio-group>
            </a-form-item>
            <a-form-item>
              <a-button type="primary" @click="handleSave">保存</a-button>
            </a-form-item>
          </a-form>
        </a-card>
      </a-col>

      <!-- 统计卡片 -->
      <a-col :xs="24" :lg="8">
        <a-card title="统计数据">
          <a-statistic title="累计接诊数" :value="1234" />
          <a-divider />
```

## 3. 组件选择决策树

### 需要显示列表数据?

- 是 → 使用 `a-table` (表格) 或 `a-list` (列表)
- 否 → 继续

### 需要卡片容器?

- 是 → 使用 `a-card`
- 否 → 继续

### 需要表单输入?

- 文本输入 → `a-input`
- 多行文本 → `a-textarea`
- 下拉选择 → `a-select`
- 单选 → `a-radio-group`
- 多选 → `a-checkbox-group`
- 日期选择 → `a-date-picker`
- 数字输入 → `a-input-number`

### 需要对话框?

- 表单对话框 → `a-modal`
- 确认对话框 → `a-popconfirm`
- 侧边抽屉 → `a-drawer`

### 需要显示状态?

- 标签 → `a-tag`
- 徽标 → `a-badge`
- 按钮 → `a-button` (不同类型)

### 需要导航?

- 侧边栏 → `a-layout-sider` + `a-menu`
- 下拉菜单 → `a-dropdown`
- 标签页 → `a-tabs`
- 面包屑 → `a-breadcrumb`

- [ ] 颜色是否符合设计系统?

- [ ] 按钮操作是否清晰?

- [ ] 状态反馈是否明确?

### 数据检查

- [ ] 数据来源是否明确?
- [ ] 数据结构是否正确?
- [ ] API 调用是否合理?

## 6. 示例: 生成"机构管理"页面 UI 图

## 7. 常见问题

### Q: 如何确定使用哪个组件?

A: 参考 `06-组件库详解.md` 中的组件说明和示例，或查阅 Ant Design Vue 官方文档。

### Q: 如何确定数据结构?

A: 参考 `08-数据层与状态管理.md` 中的类型定义。

### Q: 如何确定样式?

A: 参考 `09-UI设计规范.md` 中的设计系统。

### Q: 如何处理响应式设计?

A: 使用 Ant Design Vue 的栅格系统 (`a-row`, `a-col`)，参考 `09-UI设计规范.md`。

### Q: 如何确定页面布局?

A: 参考对应模块的文档 (02-05) 中的布局结构图。

A: 使用 `a-config-provider` 配合 `theme.darkAlgorithm`，参考 `09-UI设计规范.md` 第 11 节。

### Q: 如何实现国际化?

A: 使用 Vue I18n 配合 Ant Design Vue 的 locale 配置，参考 `09-UI设计规范.md` 第 12 节。
