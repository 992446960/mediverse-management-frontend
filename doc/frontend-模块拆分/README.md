## 🎯 核心概念

### 技术栈

- **框架**: Next.js 16.1.6 (App Router)
- **UI 库**: shadcn/ui + Tailwind CSS 4
- **状态管理**: Zustand 5.0.11
- **图标**: Lucide React
- **表单**: React Hook Form + Zod

### 模块划分

1. **仪表盘**: 统计数据展示
2. **聊天模块**: 数字医生对话
3. **知识库**: 医学知识搜索
4. **工作台**: 个人/科室/机构三级
5. **管理模块**: 系统管理功能

### 权限层级

```
个人 (Personal) → 科室 (Department) → 机构 (Organization)
```

### 分身类型

- **通用分身** (general): 机构级别
- **专科分身** (specialist): 科室级别
- **专家分身** (expert): 个人级别
