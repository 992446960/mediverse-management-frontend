# Mediverse Frontend - 项目概览

## 📋 基本信息

**项目名称**: Mediverse Management  
**项目描述**: 数字医生与知识库管理平台  
**技术栈**:

- Next.js 16.1.6 (App Router)
- React 19.2.3
- TypeScript 5
- shadcn/ui + Tailwind CSS 4
- Zustand 5.0.11 (状态管理)
- Lucide React (图标库)
- React Hook Form + Zod (表单处理)

## 🏗️ 项目结构

```
frontend/
├── app/                    # Next.js App Router 页面
│   ├── layout.tsx         # 根布局（包含侧边栏）
│   ├── page.tsx           # 仪表盘首页
│   ├── login/             # 登录模块
│   ├── chat/              # 数字医生对话模块
│   ├── knowledge-base/    # 知识库搜索模块
│   ├── my/                # 个人工作台
│   ├── dept/              # 科室工作台
│   ├── org/               # 机构工作台
│   └── admin/             # 系统管理模块
├── components/            # 可复用组件
│   ├── ui/               # shadcn/ui 基础组件（30+ 组件）
│   ├── chat/             # 聊天相关业务组件
│   ├── app-sidebar.tsx   # 主应用侧边栏
│   ├── chat-sidebar.tsx  # 对话历史侧边栏
│   └── kb-sidebar.tsx    # 知识库搜索历史侧边栏
├── lib/                  # 工具函数和 API 客户端
│   ├── api.ts           # Mock API 客户端
│   └── utils.ts         # 工具函数
├── types/               # TypeScript 类型定义
│   └── index.ts        # 所有类型定义
├── store/              # Zustand 状态管理
│   └── chatStore.ts   # 聊天会话状态
└── hooks/             # 自定义 React Hooks
    └── use-mobile.ts # 移动设备检测
```

## 🎨 设计系统

**UI 组件库**: shadcn/ui (基于 Radix UI)  
**样式方案**: Tailwind CSS 4  
**主题**: 支持深色模式 (next-themes)  
**字体**: Geist Sans + Geist Mono

**颜色系统**:

- Primary: 主色调
- Secondary: 次要色调
- Muted: 柔和背景色
- Accent: 强调色
- Destructive: 危险操作色

## 📐 布局结构

**根布局** (`app/layout.tsx`):

```
┌─────────────────────────────────────┐
│  Sidebar (可折叠)  │  Main Content  │
│                    │                 │
│  - Team Switcher   │  - Header Bar   │
│  - Nav Main        │  - Page Content │
│  - Nav Admin       │                 │
│  - Nav User        │                 │
└─────────────────────────────────────┘
```

**特殊布局**:

- 聊天模块: 左侧对话历史 + 右侧聊天界面
- 知识库: 左侧搜索历史 + 右侧搜索/结果界面
