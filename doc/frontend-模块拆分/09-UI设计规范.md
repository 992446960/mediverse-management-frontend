# UI 设计规范

## 1. 设计系统概览

### 基础框架

- **UI 组件库**: Ant Design Vue 4.x
- **样式方案**: CSS Variables + 原子化 CSS
- **图标库**: @ant-design/icons-vue
- **字体**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial

### 设计原则

- 简洁明了
- 一致性
- 可访问性
- 响应式设计
- 支持明暗主题切换
- 支持国际化（中英文）

### 语义化颜色使用

| 用途 | 颜色变量                 | 说明               |
| ---- | ------------------------ | ------------------ |
| 成功 | `--color-success`        | 成功状态、今日数据 |
| 警告 | `--color-warning`        | 警告信息、知识引用 |
| 信息 | `--color-info`           | 信息提示           |
| 危险 | `--color-error`          | 错误、删除操作     |
| 主要 | `--color-primary`        | 主要操作、强调内容 |
| 次要 | `--color-text-secondary` | 次要文本、辅助信息 |

## 5. 组件样式规范

### Button (按钮)

**类型**:

```vue
<!-- 主要按钮 -->
<a-button type="primary">保存</a-button>

<!-- 默认按钮 -->
<a-button>取消</a-button>

<!-- 虚线按钮 -->
<a-button type="dashed">草稿</a-button>

<!-- 文本按钮 -->
<a-button type="text">查看详情</a-button>

<!-- 链接按钮 -->
<a-button type="link">了解更多</a-button>

<!-- 危险按钮 -->
<a-button danger>删除</a-button>
```

**尺寸**:

```vue
<a-button size="small">小按钮</a-button>
<a-button size="middle">默认按钮</a-button>
<a-button size="large">大按钮</a-button>
```

**图标按钮**:

```vue
<a-button type="primary" :icon="h(PlusOutlined)">新增</a-button>
<a-button :icon="h(SearchOutlined)" />
```

### Card (卡片)

**标准结构**:

```vue
<a-card title="标题" :extra="extra">
  <template #extra>
    <a-button type="link">更多</a-button>
  </template>
  <p>卡片内容</p>
</a-card>
```

**样式规范**:

- 圆角: `borderRadius` token (默认 6px)
- 边框: `colorBorder` token
- 阴影: `boxShadowCard` token
- 背景: `colorBgContainer` token

### Table (表格)

**基础用法**:

```vue
<a-table
  :columns="columns"
  :data-source="dataSource"
  :pagination="pagination"
  :loading="loading"
  row-key="id"
>
  <template #bodyCell="{ column, record }">
    <template v-if="column.key === 'action'">
      <a-space>
        <a-button type="link" size="small">编辑</a-button>
        <a-button type="link" size="small" danger>删除</a-button>
      </a-space>
    </template>
  </template>
</a-table>
```

**样式规范**:

- 表头背景: `colorBgContainer`
- 行悬停: `colorBgTextHover`
- 边框: `colorBorder`
- 单元格内边距: 16px 24px

### Tag (标签)

**颜色变体**:

```vue
<!-- 默认 -->
<a-tag>默认</a-tag>

<!-- 预设颜色 -->
<a-tag color="success">成功</a-tag>
<a-tag color="processing">处理中</a-tag>
<a-tag color="error">错误</a-tag>
<a-tag color="warning">警告</a-tag>
<a-tag color="default">默认</a-tag>
```

**使用场景**:

- 状态标签: 启用/禁用
- 角色标签: 系统管理员/普通用户
- 类型标签: 循证/规则/经验

### Modal (对话框)

**标准结构**:

```vue
<a-modal
  v-model:open="visible"
  title="标题"
  :width="600"
  @ok="handleOk"
  @cancel="handleCancel"
>
  <a-form :model="formData" :label-col="{ span: 6 }">
    <a-form-item label="名称" name="name">
      <a-input v-model:value="formData.name" />
    </a-form-item>
  </a-form>
</a-modal>
```

**尺寸**:

- 小: `width="520"`
- 中: `width="600"`
- 大: `width="800"`
- 全屏: `:fullscreen="true"`

### Drawer (抽屉)

**标准结构**:

```vue
<a-drawer v-model:open="visible" title="标题" :width="600" placement="right">
  <p>抽屉内容</p>
</a-drawer>
```

## 6. 布局规范

### 页面布局

**标准页面结构**:

```vue
<template>
  <div class="page-container">
    <!-- 页面头部 -->
    <div class="page-header">
      <h2 class="page-title">页面标题</h2>
      <a-space>
        <a-button type="primary">操作按钮</a-button>
      </a-space>
    </div>

    <!-- 页面内容 -->
    <a-card>
      <!-- 内容区域 -->
    </a-card>
  </div>
</template>

<style scoped>
.page-container {
  padding: 24px;
}

.page-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  margin-bottom: 24px;
}

.page-title {
  font-size: 24px;
  font-weight: 600;
  margin: 0;
}
</style>
```

### 网格布局

**响应式网格**:

```vue
<!-- 2列网格 (移动端1列，桌面端2列) -->
<a-row :gutter="[16, 16]">
  <a-col :xs="24" :md="12">
    <a-card>卡片1</a-card>
  </a-col>
  <a-col :xs="24" :md="12">
    <a-card>卡片2</a-card>
  </a-col>
</a-row>

<!-- 4列网格 (移动端1列，平板2列，桌面4列) -->
<a-row :gutter="[16, 16]">
  <a-col :xs="24" :sm="12" :lg="6" v-for="item in 4" :key="item">
    <a-card>卡片{{ item }}</a-card>
  </a-col>
</a-row>
```

### 侧边栏布局

**左侧边栏 + 右侧内容**:

```vue
<a-layout style="min-height: calc(100vh - 64px)">
  <!-- 左侧边栏 -->
  <a-layout-sider width="256" theme="light">
    <a-card>
      <a-tree :tree-data="treeData" />
    </a-card>
  </a-layout-sider>

  <!-- 右侧内容 -->
  <a-layout-content style="padding: 24px">
    <a-card>
      <!-- 主要内容 -->
    </a-card>
  </a-layout-content>
</a-layout>
```

## 7. 响应式设计

### 断点

| 断点   | 类前缀 | 最小宽度 |
| ------ | ------ | -------- |
| 超小屏 | `xs`   | < 576px  |
| 小屏   | `sm`   | ≥ 576px  |
| 中屏   | `md`   | ≥ 768px  |
| 大屏   | `lg`   | ≥ 992px  |
| 超大屏 | `xl`   | ≥ 1200px |
| 超超大 | `xxl`  | ≥ 1600px |

### 响应式栅格

```vue
<a-col :xs="24" :sm="12" :md="8" :lg="6" :xl="4">
  <!-- 不同屏幕尺寸显示不同列数 -->
</a-col>
```

## 8. 交互状态

### 悬停状态

Ant Design Vue 组件自动处理悬停状态，使用 token 配置：

```typescript
{
  colorPrimaryHover: '#40a9ff',
  colorLinkHover: '#40a9ff'
}
```

### 焦点状态

```typescript
{
  colorPrimaryBorder: '#40a9ff',
  controlOutline: 'rgba(24, 144, 255, 0.2)'
}
```

### 禁用状态

```vue
<a-button disabled>禁用按钮</a-button>
<a-input disabled />
<a-select disabled />
```

### 加载状态

```vue
<a-button :loading="loading">加载中...</a-button>
<a-spin :spinning="loading">
  <a-card>内容</a-card>
</a-spin>
```

<!-- 中等 16px -->

<!-- 大图标 20px -->

### 图标位置

**按钮中的图标**:

```vue
<a-button type="primary">
  <template #icon><PlusOutlined /></template>

</a-button>
```

**标题中的图标**:

```vue
<h2>
  <MessageSquareOutlined style="margin-right: 8px" />
  标题
</h2>
```

## 10. 动画效果

### 过渡动画

Ant Design Vue 内置动画，通过 `motion` token 配置：

```typescript
{
  motionDurationFast: '0.1s',
  motionDurationMid: '0.2s',
  motionDurationSlow: '0.3s'
}
```

### 自定义过渡

```vue
<transition name="fade">
  <div v-if="visible">内容</div>
</transition>

<style>
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s;
}

.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
</style>
```

## 11. 主题切换

### 实现方式

```vue
<template>
  <a-config-provider :theme="themeConfig">
    <router-view />
  </a-config-provider>
</template>

<script setup lang="ts">
import { computed } from 'vue'
import { theme } from 'ant-design-vue'
import { useThemeStore } from '@/stores/theme'

const themeStore = useThemeStore()

const themeConfig = computed(() => ({
  algorithm:
    themeStore.actualTheme === 'dark'
      ? theme.darkAlgorithm
      : theme.defaultAlgorithm,
  token: {
    colorPrimary: '#1890ff',
    borderRadius: 6
    // ... 更多 token 配置
  }
}))
</script>
```

### 主题切换组件

参考 `前端模块设计.md` 中的 `ThemeSwitcher` 组件实现。

## 12. 国际化

### 配置方式

```vue
<template>
  <a-config-provider :locale="antdLocale">
    <router-view />
  </a-config-provider>
</template>

<script setup lang="ts">
import { computed } from 'vue'
import { useLocaleStore } from '@/stores/locale'

const localeStore = useLocaleStore()
const antdLocale = computed(() => localeStore.getAntdLocale())
</script>
```

### 文本国际化

```vue
<template>
  <a-button>{{ t('common.save') }}</a-button>
</template>

<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { t } = useI18n()
</script>
```
