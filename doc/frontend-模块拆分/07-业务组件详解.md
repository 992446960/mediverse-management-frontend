# 业务组件详解

## 1. 导航相关组件

### AppSidebar (主应用侧边栏)

**文件**: `frontend/components/app-sidebar.tsx`  
**用途**: 应用主侧边栏，包含导航菜单

**组成部分**:

1. **Header**: TeamSwitcher (团队/机构切换器)
2. **Content**:
   - NavMain (平台功能菜单)
   - NavMain (系统管理菜单)
3. **Footer**: NavUser (用户菜单)

**导航数据结构**:

```typescript
{
  navMain: [
    {
      title: "仪表盘",
      url: "/",
      icon: Home,
      isActive: true
    },
    {
      title: "我的工作台",
      url: "/my",
      icon: User,
      items: [
        { title: "我的分身", url: "/my/avatar" },
        { title: "我的文件", url: "/my/files" },
        { title: "我的知识卡", url: "/my/knowledge-cards" }
      ]
    }
    // ...更多菜单项
  ],
  navAdmin: [
    { title: "机构管理", url: "/admin/organizations", icon: Building2 },
    { title: "科室管理", url: "/admin/departments", icon: Frame },
    // ...更多管理菜单
  ]
}
```

**布局**:

```
┌─────────────────┐
│ TeamSwitcher    │ ← Header
├─────────────────┤
│ 平台功能        │
│ ▶ 仪表盘        │
│ ▼ 我的工作台    │
│   - 我的分身    │
│   - 我的文件    │
│ ▶ 科室工作台    │
│ ...             │ ← Content
├─────────────────┤
│ 系统管理        │
│ ▶ 机构管理      │
│ ▶ 用户管理      │
│ ...             │
├─────────────────┤
│ NavUser         │ ← Footer
└─────────────────┘
```

### NavMain (导航菜单项)

**文件**: `frontend/components/nav-main.tsx`  
**用途**: 渲染导航菜单项（支持嵌套）

**Props**:

```typescript
{
  items: {
    title: string,
    url: string,
    icon?: LucideIcon,
    isActive?: boolean,
    items?: { title: string, url: string }[]
  }[],
  label?: string
}
```

**特性**:

- 支持单层嵌套（父菜单 + 子菜单）
- 使用 Collapsible 实现展开/折叠
- 支持图标显示
- 支持活跃状态高亮

**渲染逻辑**:

- 无子菜单: 渲染简单链接
- 有子菜单: 渲染可折叠菜单

## 2. 聊天相关组件

### ChatSidebar (对话历史侧边栏)

**文件**: `frontend/components/chat-sidebar.tsx`  
**用途**: 显示和管理对话历史

**功能**:

- 显示对话列表（按日期分组）
- 新建对话
- 重命名对话
- 删除对话
- 切换活跃对话

**状态管理**: 使用 `useChatStore` (Zustand)

**布局**:

```
┌─────────────────┐
│ [新建对话] 按钮 │
├─────────────────┤
│ 今天            │
│ 💬 高血压咨询   │
│ 💬 感冒分析     │
├─────────────────┤
│ 昨天            │
│ 💬 体检报告     │
│ 💬 饮食建议     │
├─────────────────┤
│ 过去 7 天       │
│ 💬 疫苗计划     │
└─────────────────┘
```

**交互**:

- 点击对话: 跳转到对话详情页
- 悬停显示: 更多操作按钮（重命名、删除）

---

### AvatarSelector (分身选择器)

**文件**: `frontend/components/chat/avatar-selector.tsx`  
**用途**: 选择数字医生分身开始对话

**Props**:

```typescript
{
  avatars: Avatar[],
  organizations: Organization[],
  departments: Department[]
}
```

**功能**:

- 按类型分组显示分身（通用/专科/专家）
- 显示分身信息（头像、名称、简介、标签）
- 点击分身开始新对话

**布局**:

```
┌─────────────────────────────────┐
│ 通用分身                        │
│ ┌─────────┐ ┌─────────┐        │
│ │[头像]   │ │[头像]   │        │

│ │简介     │ │简介     │        │
│ │[标签]   │ │[标签]   │        │
│ └─────────┘ └─────────┘        │
├─────────────────────────────────┤
│ 专科分身                        │
│ ┌─────────┐ ┌─────────┐        │
│ │...      │ │...      │        │
│ └─────────┘ └─────────┘        │
└─────────────────────────────────┘
```

## 组件使用示例

### 创建带侧边栏的页面布局

```tsx
// app/chat/layout.tsx
import { ChatSidebar } from '@/components/chat-sidebar'

export default function ChatLayout({ children }) {
  return (
    <div className='flex h-[calc(100vh-4rem)] overflow-hidden'>
      <ChatSidebar />
      <div className='flex-1 flex overflow-hidden'>{children}</div>
    </div>
  )
}
```

### 使用分身选择器

```tsx
// app/chat/page.tsx
import { AvatarSelector } from '@/components/chat/avatar-selector'
import { api } from '@/lib/api'

export default async function ChatPage() {
  const [avatars, organizations, departments] = await Promise.all([
    api.getAvatars(),
    api.getOrganizations(),
    api.getDepartments()
  ])

  return (
    <AvatarSelector
      avatars={avatars}
      organizations={organizations}
      departments={departments}
    />
  )
}
```
